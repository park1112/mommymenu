# MommyMenu 메인 페이지 상세 기획서

## 1. 페이지 개요

### 1.1 목적
- **신규 방문자**: 서비스 가치 전달 및 회원가입 유도
- **기존 회원**: 빠른 로그인 및 대시보드 접근
- **파트너**: B2B 솔루션 안내

### 1.2 핵심 목표
- 3초 내 핵심 가치 전달
- 30초 내 차별화 요소 이해
- 3분 내 회원가입 완료

## 3. 섹션별 상세 기획

### 3.1 Navigation Bar (고정형)

#### 구성 요소
```
[로고] [서비스 소개 ▼] [요금제] [기업 고객] [블로그] [전문가 네트워크]     [로그인] [무료 시작하기]
```

#### 기능 상세
- **로고**: 클릭 시 메인 페이지 최상단 이동
- **서비스 소개 드롭다운**:
  - 핵심 기능
  - 과학적 근거
  - 성공 사례
  - 의료진 네트워크
- **스크롤 반응**: 배경 블러 효과, 높이 축소 (80px → 60px)
- **모바일**: 햄버거 메뉴로 전환

## 5. 성능 최적화

### 5.1 로딩 전략
- **Above the fold**: 2초 내 로드
- **Lazy loading**: 이미지, 영상
- **Progressive enhancement**: 핵심 → 부가 기능

### 5.2 이미지 최적화
- **포맷**: WebP (폴백: JPEG)
- **반응형**: srcset 활용
- **압축**: 품질 85% 유지

## 6. A/B 테스트 포인트

### 6.1 테스트 요소
- **헤드라인**: 감성적 vs 기능적
- **CTA 문구**: "시작하기" vs "체험하기"
- **가격 표시**: 월간 vs 연간
- **증거 제시**: 숫자 vs 스토리

### 6.2 성공 지표
- **전환율**: 방문자 → 회원가입
- **이탈률**: 섹션별 스크롤 깊이
- **체류 시간**: 평균 3분 이상
- **클릭률**: CTA 버튼 클릭
