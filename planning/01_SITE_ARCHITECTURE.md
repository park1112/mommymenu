# MommyMenu 사이트 아키텍처 및 정보 구조 설계

## 1. 서비스 개요

### 1.1 서비스명
**MommyMenu (맘미뉴)** - 임산부 맞춤 AI 영양 관리 플랫폼

### 1.2 핵심 가치 제안
- **과학적 개인화**: 영양유전체학 기반 맞춤 식단
- **한국형 솔루션**: 전통 산후조리와 현대 영양학의 융합
- **실시간 적응**: AI 기반 즉각적 식단 조정
- **통합 건강 관리**: CGM 연동, 의료진 협업

### 1.3 타겟 사용자
- **Primary**: 20-40대 임신 여성 (초산/경산)
- **Secondary**: 임신 준비 중인 여성
- **Tertiary**: 배우자 및 가족

### 2.2 사용자 여정별 페이지 구조

#### A. 온보딩 플로우
```
회원가입 → 임신 정보 입력 → 건강 상태 체크 → 
선호도 설정 → 유전자 검사 안내(선택) → 첫 식단 생성
```

#### B. 일일 사용 플로우
```
대시보드 → 오늘의 식단 확인 → 식사 기록 → 
영양 현황 체크 → 다음 식사 추천 → 피드백
```

#### C. 주간 계획 플로우
```
주간 목표 설정 → 식단 계획 생성 → 장보기 리스트 → 
레시피 준비 → 실행 및 추적 → 주간 리포트
```

## 3. 핵심 페이지 구성

### 3.1 랜딩 페이지
- **Hero Section**: 핵심 가치 제안
- **서비스 특징**: 4대 차별화 요소
- **과학적 근거**: 연구 데이터 시각화
- **사용자 후기**: 실제 경험담
- **가격 플랜**: B2C/B2B 옵션
- **CTA**: 무료 체험 시작

### 3.2 대시보드 (홈)
- **임신 주차 위젯**: 현재 상태 요약
- **오늘의 식단 카드**: 3끼 미리보기
- **영양 현황 게이지**: 주요 영양소 5종
- **건강 지표**: 체중, 혈당, 활동량
- **AI 추천**: 맞춤 팁 & 레시피
- **퀵 액션**: 식사 기록, 증상 입력

### 3.3 식단 생성기
- **Step 1**: 현재 상태 입력 (증상, 기분)
- **Step 2**: 식사 유형 선택 (아침/점심/저녁/간식)
- **Step 3**: 선호도 반영 (음식 종류, 조리 시간)
- **Step 4**: AI 식단 생성 (3-5개 옵션)
- **Step 5**: 선택 및 커스터마이징
- **Step 6**: 레시피 상세 & 영양 정보

### 3.4 영양 분석 센터
- **대시보드 뷰**: 종합 영양 스코어
- **영양소별 상세**: 30+ 영양소 추적
- **트렌드 분석**: 주간/월간 변화
- **경고 알림**: 부족/과잉 영양소
- **개선 제안**: AI 기반 조언

### 3.5 커뮤니티 허브
- **피드**: 실시간 게시물
- **그룹**: 임신 주차별, 관심사별
- **전문가 코너**: 인증된 전문가 콘텐츠
- **라이브 세션**: 실시간 강의/상담

## 4. 모바일 우선 설계

### 4.1 모바일 네비게이션
```
Bottom Tab Navigation (5개)
├── 홈 (대시보드)
├── 식단
├── 기록
├── 커뮤니티
└── 내 정보
```

### 4.2 모바일 특화 기능
- **원터치 식사 기록**: 사진 촬영 → AI 인식
- **음성 입력**: 증상, 메모 음성 기록
- **위젯**: 홈 화면 식단 위젯
- **알림**: 식사 시간, 영양제 복용
- **오프라인 모드**: 기본 기능 사용 가능

## 5. 기술 스택 고려사항

### 5.1 프론트엔드
- **Framework**: Next.js 14 (App Router)
- **UI Library**: Tailwind CSS + Radix UI
- **State Management**: Zustand
- **Data Fetching**: TanStack Query
- **PWA**: 오프라인 지원

### 5.2 백엔드
- **API**: Next.js API Routes + tRPC
- **Database**: Supabase (PostgreSQL)
- **AI/ML**: OpenAI API + Custom Models
- **File Storage**: Supabase Storage
- **Real-time**: Supabase Realtime

### 5.3 특수 기능
- **CGM Integration**: Bluetooth SDK
- **Computer Vision**: TensorFlow.js
- **Analytics**: Vercel Analytics
- **Push Notifications**: OneSignal
