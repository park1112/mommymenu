# MommyMenu 통합 기획서 - 마스터 플랜

## 📋 문서 개요

본 문서는 **MommyMenu (맘미뉴)** - 임산부 맞춤 AI 영양 관리 플랫폼의 종합 기획서입니다. 
plen.md의 비즈니스 전략을 바탕으로 실제 웹사이트 구현을 위한 상세 설계를 포함합니다.

├── 05_NUTRITION_ANALYSIS.md    # 영양 분석 페이지 기획

## 🎯 프로젝트 비전 & 목표

### 핵심 비전
> "과학과 문화가 만나는 임산부 영양 관리의 혁신"

### 차별화 전략
1. **영양유전체학 기반 개인화** - MTHFR 등 유전자 변이 고려
2. **AI 실시간 적응** - 입덧, 갈망 등 실시간 상태 반영
3. **한국형 솔루션** - 전통 산후조리와 현대 영양학 융합
4. **의료진 연계** - 산부인과, 영양사 실시간 협진

### 주요 목표 지표
- **사용자 참여**: 월 30회 이상 앱 사용
- **영양 개선**: 사용자 영양 점수 평균 85점 이상
- **만족도**: NPS 70+ 달성
- **성장률**: 연 150% 사용자 증가

## 🏗️ 시스템 아키텍처 요약

### 기술 스택
```yaml
Frontend:
  Framework: Next.js 14 (App Router)
  UI: Tailwind CSS + Radix UI
  State: Zustand + TanStack Query
  PWA: Service Worker + Web App Manifest

Backend:
  API: Next.js API Routes + tRPC
  Database: Supabase (PostgreSQL)
  Storage: Supabase Storage
  Real-time: Supabase Realtime

AI/ML:
  LLM: OpenAI GPT-4 + Custom Models
  Vision: TensorFlow.js (음식 인식)
  Analytics: Custom ML Pipeline

External:
  CGM: Bluetooth SDK Integration
  Payments: Stripe + Portone
  Push: OneSignal
```

### 주요 기능 모듈
1. **사용자 관리** - 회원가입, 프로필, 임신 정보
2. **AI 식단 생성** - 개인화 알고리즘, 레시피 매칭
3. **영양 분석** - 실시간 추적, 트렌드 분석
4. **커뮤니티** - 그룹, Q&A, 전문가 상담
5. **건강 연동** - CGM, 체중계, 활동량계
6. **의료진 연계** - EMR 연동, 리포트 공유

## 📱 사용자 여정 설계

### 신규 사용자 온보딩 (5분)
```
1. 회원가입 (1분)
   └─ 소셜 로그인 또는 이메일

2. 임신 정보 입력 (2분)
   └─ 주차, 체중, 기존 건강 상태

3. 선호도 설정 (1분)
   └─ 음식 취향, 알레르기, 제한사항

4. 유전자 검사 안내 (30초)
   └─ 선택사항, 나중에 가능

5. 첫 식단 생성 (30초)
   └─ AI 추천 → 즉시 체험
```

### 일일 사용 패턴 (5-10분)
```
아침 (2분):
├─ 대시보드 확인
├─ 오늘 식단 체크
└─ 컨디션 입력

점심/저녁 (3분):
├─ 식사 기록 (사진 촬영)
├─ 영양 현황 확인
└─ 다음 식사 추천

취침 전 (2분):
├─ 일일 리뷰
├─ 내일 계획
└─ 커뮤니티 둘러보기
```
