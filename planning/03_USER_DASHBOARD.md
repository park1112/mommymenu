# MommyMenu 사용자 대시보드 상세 기획서

## 1. 대시보드 개요

### 1.1 목적
- **일일 허브**: 모든 주요 기능의 시작점
- **현황 파악**: 한눈에 보는 건강 상태
- **빠른 실행**: 원클릭 주요 작업 수행

### 1.2 사용자 시나리오
- **아침 체크**: 오늘의 식단 확인, 증상 입력
- **식사 시간**: 식사 기록, 다음 메뉴 확인
- **저녁 리뷰**: 일일 영양 평가, 내일 계획

## 3. 위젯별 상세 기능

### 3.1 임신 상태 위젯

#### 표시 정보
```javascript
{
  "주차": "23주 3일",
  "태아": {
    "크기": "약 28.9cm",
    "체중": "약 501g",
    "발달": "청각 발달, 폐 성숙 시작",
    "비유": "큰 망고 크기"
  },
  "엄마": {
    "체중증가": "+5.2kg",
    "권장범위": "4.5-6.8kg",
    "상태": "정상"
  }
}
```

#### 인터랙션
- **클릭**: 임신 주차별 상세 가이드
- **스와이프**: 이전/다음 주 미리보기
- **길게 누르기**: 주차 계산기

### 3.2 오늘의 식단 타임라인

#### 시각적 표현
```
아침 (7:00-9:00)    점심 (12:00-14:00)    저녁 (18:00-20:00)
     ●                    ○                     ○
  [완료]               [예정]                [대기]
  
  간식1 (10:30)     간식2 (15:30)      야식 (21:00)
      ●                  ○                 ×
   [완료]            [진행중]           [건너뜀]
```

#### 상태별 표시
- **완료**: 초록색 체크, 영양 정보 표시
- **진행중**: 주황색 점멸, 타이머 표시
- **예정**: 회색, 알림 설정 가능
- **놓침**: 빨간색, 대체 제안

## 4. 개인화 기능

### 4.1 위젯 커스터마이징
- **드래그 앤 드롭**: 위젯 위치 변경
- **크기 조절**: S/M/L 사이즈
- **숨기기/보이기**: 위젯 선택

### 4.2 테마 설정
```
라이트 모드 | 다크 모드 | 시스템 설정
  ☀️          🌙         🔄
```

### 4.3 알림 설정
- 식사 시간 알림
- 영양제 복용 알림
- 물 마시기 알림
- 운동 알림

## 5. 데이터 시각화

### 5.1 차트 타입
- **라인 차트**: 체중, 혈당 추이
- **도넛 차트**: 영양소 비율
- **바 차트**: 주간 비교
- **히트맵**: 월간 패턴

### 5.2 인터랙티브 요소
- **호버**: 상세 수치 표시
- **클릭**: 확대/상세 보기
- **드래그**: 기간 선택
- **핀치**: 줌 인/아웃

## 6. 실시간 업데이트

- 이상 수치 감지
- 전문가 답변

## 7. 성능 최적화

### 7.1 로딩 전략
- **스켈레톤 스크린**: 1초 내 표시
- **프로그레시브 로딩**: 중요도 순
- **캐싱**: 오프라인 지원

### 7.2 데이터 최적화

- **상세 데이터**: 온디맨드
- **압축**: gzip 적용

## 8. 접근성 기능

### 8.1 시각 접근성
- **고대비 모드**: WCAG AAA

### 8.2 조작 접근성
- **큰 터치 영역**: 최소 48px
